About SIBAT

SIBAT is a bilingual statistics dictionary website developed as a school project for educational purposes. The website aims to help students understand commonly used statistical and data science terms through simple English and Filipino definitions.

The project was created to make learning statistics more accessible, understandable, and student-friendly through an interactive and organized web platform.

Objectives

To provide bilingual definitions of statistical terms

To help students understand statistics more easily

To organize statistical concepts into categories

To create an interactive educational website

To support learning using English and Filipino language

Purpose of the Website

Many students find statistics difficult because of
technical and unfamiliar terminology. This website
serves as a learning aid that simplifies these terms
using clear explanations and bilingual translations.

The platform also promotes easier navigation and
faster access to important statistical concepts.
